# Spinner Pro Mobile

React Native (Expo) ile mobil odakli Spinner Pro MVP.

## Kurulum

```bash
npm install
npx expo start
```

Telefonla test:

1. Telefona Expo Go kur.
2. Bilgisayar ve telefon ayni Wi-Fi aginda olsun.
3. `npx expo start` ciktisindaki QR kodu Expo Go ile tara.

## Ozellikler

- Dikey swipe ile spinner donusu
  - asagi swipe: saga donus
  - yukari swipe: sola donus
  - ayni yone devam: hizlanma
  - ters yone swipe: once yavaslama
- `SPINS` (oturum) ve `TOTAL` (tum oturumlar) sayaclari
- Canli hiz metre ibresi
- Spinner tema secimi
- Arka plan secimi
- Basit ayarlar paneli (surtunme/hassasiyet)
